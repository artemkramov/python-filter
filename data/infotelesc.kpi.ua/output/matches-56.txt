MODEL FOR EFFICIENT ALLOCATION OF NETWORK FUNCTIONS IN HYBRID ENVIRONMENT
Mariia Skulysh, Larisa Globa, Svitlana Sulima
<p><strong>Background.</strong>To ensure the required quality of experience and optimal use of network operator's resources in the current&nbsp;conditions telecommunication operator can deploy services using the concept of Network Functions Virtualization (NFV), that&nbsp;has fundamental differences from traditional dedicated hardware. To achieve the expected benefits of NFV, physical resources&nbsp;must be used effectively. This requires effective algorithms to determine onto which physical resources network functions are&nbsp;allocated.<br><strong>Objective.</strong>The aim of the paper is to improve the efficiency of mobile network through optimal resource allocation in hybrid&nbsp;data center environment.<br><strong>Methods.</strong>Analysis of all known publications devoted to virtualization of network functions of mobile network has shown&nbsp;the modeling approach to resource allocation and also has shown the absence of decisions on important issues of this process&nbsp;(performance of functional blocks, heterogeneous environment).<br><strong>Results.</strong>An analytical approach to model and investigate the resource allocation of network functions on a telecommunication&nbsp;operator’s network is proposed.<br><strong>Conclusions.</strong>To model the resource allocation for virtualized network functions a technique of virtual network embedding&nbsp;can be used, while changing the constraints in which you can get the results for a variety of deployment scenarios. In such way,&nbsp;adding a number of constraints to the classical formulation of the problem, we can take into account the performance of the<br>network functions and heterogeneity (hybridity) of the deployment environment.</p></div><br></div><div><h4>Keywords</h4><br><div>NFV; cloud computing; resource allocation; virtual network embedding
