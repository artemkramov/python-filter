/div><div>We present here a highly portable and easy-to-use gene annotation system CEL (Computational Environment for<br><br>annotation of Legume genomes) that can be used to annotate any type of genomic sequence -- from BAC ends to<br><br>complete chromosomes. CEL’s core engine is modular and hierarchically organized with an open-source structure,<br><br>permitting maximum customization -- users can assemble an individualized annotation pipeline by selecting<br><br>computational components that best suit their annotation needs. The tool is designed to speed up genomic analyses<br><br>and features an algorithm that substitutes for a biologist’s expertise at various steps of gene structure prediction.<br><br>This allows more complete automation of the labor-intensive and time-consuming annotation process.<br><br>The system collects and prioritizes multiple sources of de novo gene predictions and gene expression evidence<br><br>according to the confidence value of underlying supporting evidence, as a result producing high-quality genemodel<br><br>sets. The data produced by CEL pipeline is suitable for direct visualization in any genome browser tool that<br><br>supports GFF annotation format (e.g. Apollo, Artemis, Genome Browser etc.). This provides an easy means to<br><br>view and edit individual contigs and BACs using just mouse’s clicks and drag-and-drop features. Finally, we show<br><br>that CEL produces accurate annotations for novel draft genomes,of low quality or mostly non-existent, as in<br><br>the case of narrow-leafed lupin where the training-data are limited.<
